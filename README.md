### House Price Prediction System



#### Project Overview



###### This project is a House Price Prediction System built using Python and basic Machine Learning techniques.

###### It predicts the price of a house based on features like area, number of bedrooms, and bathrooms.

###### 

###### The model is trained using Linear Regression, one of the simplest and most widely used algorithms for prediction tasks.

###### 

#### Objectives



* ###### Predict house prices accurately based on input features.
* ###### Understand the implementation of regression models.
* ###### Build a real-world ML project for learning and portfolio.



#### Technologies Used



* ###### Python
* ###### Pandas
* ###### NumPy
* ###### Matplotlib
* ###### Seaborn
* ###### Scikit-learn



#### Dataset

###### 

###### The project currently uses a sample dataset created within the code.

###### 

#### How It Works

###### 

###### The system follows these steps:

1. ###### Data collection (sample or CSV dataset)
2. ###### Data visualization using graphs
3. ###### Feature selection (area, bedrooms, bathrooms)
4. ###### Splitting data into training and testing sets
5. ###### Training the model using Linear Regression
6. ###### Evaluating model performance
7. ###### Predicting price based on user input



#### Model Used

###### 

###### Linear Regression

###### The model follows this equation:

###### Price = (area × weight) + (bedrooms × weight) + (bathrooms × weight) + bias

######
